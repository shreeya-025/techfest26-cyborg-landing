# 🤖 TECHFEST '26 — Chronicles of Synapse
### IIT Bombay | Cyborg-Themed Landing Page

> *"Where Biological Intelligence Meets the Machine"*

A fully responsive, cyberpunk/cyborg-themed landing page built for **IIT Bombay's Techfest 2026** submission. Featuring dynamic canvas particle networks, an interactive terminal console, augmented bionic body schematic, Web Audio API synthesizer, and premium HUD-styled glassmorphism design.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🧠 Neural Canvas | Dynamic particle-node network background that reacts to mouse movement |
| 🖥️ Core Terminal | Interactive shell emulator — type `help`, `hack`, `events`, `status`, `register` |
| 🦾 Augmentation Map | Clickable cybernetic body schematic with live diagnostic info panel |
| 🔊 Audio Synthesizer | Web Audio API-powered click/hover/success sounds (zero external assets) |
| 📡 Registration Portal | Animated form with live telemetry sync indicator and success modal |
| 🎛️ Event Matrix Filters | ROBOTICS / CODING / AEROSPACE animated grid sorting |
| 📱 Fully Responsive | Mobile-first, works across all screen sizes (320px to 4K) |

## 🎮 Terminal Commands

Once on the site, navigate to the **Core Terminal** section and try:

| Command | Action |
|---|---|
| `help` | List all available commands |
| `about` | Chronicles of Synapse narrative |
| `status` | Mainframe diagnostic telemetry |
| `events` | List all competition sub-modules |
| `register` | Jump to registration portal |
| `hack` | 🔴 Launch the codebreaker override matrix |
| `clear` | Wipe terminal cache display |
